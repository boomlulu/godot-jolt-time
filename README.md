# godot-jolt-time

Godot 4.6.3 工程，使用 Jolt Physics 物理引擎。

## 工程信息

- **引擎版本**：Godot 4.6.3
- **配置名**：JoltTIme
- **渲染方法**：Mobile
- **物理引擎**：Jolt Physics
- **Windows 渲染设备**：D3D12

## 目录结构

```
.
├── project.godot       # 工程配置
├── icon.svg            # 工程图标
├── .editorconfig
├── .gitattributes
└── .gitignore
```

## 使用

用 Godot 4.6.3 打开本目录即可。
